# Portfólio Henrique Troiano - Especialista CS & CX

Um portfólio profissional, moderno e interativo desenvolvido para Henrique Troiano, Especialista em Customer Success (CS) e Customer Experience (CX). O projeto visa apresentar sua visão estratégica, experiência e resultados reais através de uma interface web de alta performance e responsiva.

## 🚀 Funcionalidades

- **Design Premium:** Interface com tema escuro (Dark Theme), detalhes em Glassmorphism e tipografia moderna.
- **Bilingue (PT-BR / EN):** Suporte nativo a múltiplos idiomas com troca dinâmica sem recarregar a página.
- **Scroll Suave (Smooth Scrolling):** Utilização da biblioteca Lenis para uma experiência de rolagem inercial imersiva.
- **Interatividade & Animações:** Cursor reativo personalizado, efeitos de *reveal* ao rolar a página e transições suaves.
- **Contato Dinâmico:** Botões integrados para WhatsApp, LinkedIn e download de VCard.
- **Responsividade Total:** Layout adaptável para Desktop, Tablets e dispositivos móveis, com menu *off-canvas* (mobile menu).
- **Acessibilidade & SEO:** Uso de tags semânticas e Open Graph Meta Tags.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web modernas para garantir performance e máxima customização, com foco em Vanilla para aprendizado e controle total.

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização com Flexbox, CSS Grid, Custom Properties (Variáveis CSS) e animações (`@keyframes`).
- **JavaScript (ES6+):** Lógica da interface, manipulação de DOM, i18n dinâmico e controle de interações.
- **[Lenis](https://lenis.studiofreight.com/):** Biblioteca de Smooth Scroll para proporcionar navegação fluida.
- **[Phosphor Icons](https://phosphoricons.com/):** Biblioteca de ícones minimalistas e versáteis.
- **[Google Fonts](https://fonts.google.com/):** Fontes Inter e Outfit.

## 📁 Estrutura do Projeto

- `index.html`: Estrutura principal da página e importação de metadados.
- `style.css`: Todo o sistema de design, variáveis, estilos de componentes e media queries.
- `script.js`: Toda a lógica de interatividade, i18n, controle de tema e navegação fluida.
- `image/`: Diretório contendo as imagens e assets visuais utilizados no projeto.
